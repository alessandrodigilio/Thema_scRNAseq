# Samplesheets

This folder contains samplesheets documenting the FASTQ inputs used for upstream nf-core/scrnaseq processing.

- `scRNA_samplesheet_all_samples.csv`: full-cohort samplesheet template for the samples retained in the downstream analysis.
- `scRNA_samplesheet_original_4HA.csv`: original 4-sample HA samplesheet kept for traceability.

Paths are intentionally written with `~/raw_data/...` placeholders and should be adapted to the local FASTQ location before running nf-core/scrnaseq.
